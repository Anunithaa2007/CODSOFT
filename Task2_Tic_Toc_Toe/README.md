# Tic Tac Toe AI

## Description
This project is a Tic Tac Toe game developed in Python as part of the CodSoft Artificial Intelligence Internship.

## Features
- Human vs AI gameplay
- Interactive console-based game
- Winner and draw detection
- Play again option

## Technologies Used
- Python

## How to Run
1. Install Python.
2. Open the project folder.
3. Run:tic_toc_toe.py
