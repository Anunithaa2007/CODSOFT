# Rule-Based Chatbot

## Objective
Build a simple chatbot that responds to user inputs using predefined rules.

## Features
- Greets the user.
- Responds to simple questions.
- Gives a default response for unknown inputs.
- Exits when the user types "bye".

## Technologies Used
- Python
- Visual Studio Code

## How to Run
1. Open the project in Visual Studio Code.
2. Run the `chatbot.py` file.
3. Type your messages and interact with the chatbot.
4. Type `bye` to exit.
