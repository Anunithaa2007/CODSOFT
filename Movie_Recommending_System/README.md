# Movie Recommendation System

## Description
This project is a simple Movie Recommendation System developed in Python as part of an Artificial Intelligence Internship. It recommends similar movies to users based on their preferences using Content-Based Filtering.

## Features
- Content-based movie recommendations
- User-friendly console interface
- Uses TF-IDF Vectorizer
- Calculates similarity using Cosine Similarity
- Recommends the top similar movies

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
1. Install Python.
2. Install the required libraries:
   ```
   pip install pandas scikit-learn
   ```
3. Open the project folder.
4. Run:
   ```
   python recommendation.py
   ```
